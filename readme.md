# auto-crud

Полностью самописный mvc и orm. Позволяет создавать crud приложения без написания кода, либо с минимальным написанием кода.


В планах написать скрипт для генерации моделей.

### Запуск:


```shell
mv src/.env.example src/.env
```
```shell
docker-compose up -d
```
